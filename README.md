# eni (WIP)

## Commands
### track
Usage: `eni track TICKET-123 "Did some stuff"`
### note
Usage: `eni note Think about the moon`
### start
Usage: `eni start`
### stop
Usage: `eni stop`
### list 
Usage: `eni list`

## Configuration
This will place a file in your home directory `~/.eni/config.json.dist` with a sample config
